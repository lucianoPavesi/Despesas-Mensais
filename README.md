# Despesas-Mensais
Este aplicativo é um gerenciador financeiro prático e visual. Na tela inicial, você acompanha o saldo mensal, receitas e despesas. Com um toque no botão (+), registra novos lançamentos rapidamente. A lista exibe o status de pagamentos e parcelas. É a ferramenta ideal para ter controle total do seu dinheiro com muita eficiência e agilidade.
